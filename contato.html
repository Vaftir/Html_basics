Pagina de contato

Se o seu git Shell não estiver configurado com direitos administrativos apropriados (ou seja, você negar quando tentar rmdir), você pode abrir um prompt de comando (possivelmente como administrador - pressione a tecla Windows, digite 'cmd' e clique com o botão direito Prompt 'e selecione' executar como administrador) e tente os mesmos comandos.
